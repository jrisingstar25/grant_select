== Description ==

Profile Builder - Social Connect Add-On

Extends the functionality of Profile Builder by adding possibility of having multi-page registration and edit-profile forms.


NOTE:
This plugin is compatible with Profile Builder version 2.5.1 or higher


== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
3. Go to Manage Fields / Registration Forms / Edit-Profile Forms subpages and add Break Points.
